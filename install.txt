KamiCmd installation instructions

1. Download and install latest Qt Framework
   (http://qt.nokia.com/downloads)
2. Download and install latest CMake
   (http://cmake.org/cmake/resources/software.html)
3. In build directory execute cmake .. or use CMakeGUI. 
	Provide --prefix=<prefix> if you want specify install path.
	If needed, specify generator -G <generator name>
4. Build with selected tool (e.g. make for Unix Makefiles, nmake for NMake makefiles or
	open generated solution if you used Visual Studio generator)
6. Build install target
	make install
	nmake install
	or build project INSTALL
7. KamiCmd ready to use.

NOTICE: if you added or removed source files and not changed any CMakeLists.txt 
	you musr re-run CMake manually/
